LaserEyes is a comprehensive Bitcoin wallet integration library for React applications. It provides a unified interface for connecting to 14+ Bitcoin wallets, managing transactions, and interacting with Bitcoin protocols including Runes, BRC-20, Inscriptions, and Alkanes.

## Quick Start

```bash
npm install @omnisat/lasereyes
```

```tsx
import { LaserEyesProvider, useLaserEyes, MAINNET } from '@omnisat/lasereyes'

function App() {
  return (
    <LaserEyesProvider config={{ network: MAINNET }}>
      <WalletButton />
    </LaserEyesProvider>
  )
}

function WalletButton() {
  const { connect, address, connected } = useLaserEyes()

  if (connected) return <p>Connected: {address}</p>
  return <button onClick={() => connect('unisat')}>Connect Wallet</button>
}
```
